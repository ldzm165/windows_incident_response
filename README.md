# Incident Response Evidence Collection Tool

## Overview
This script automates the collection of forensic evidence from Windows systems during incident response investigations. It gathers critical system, network, security, and user information in an organized manner for analysis.

## Quick Start

### Basic Usage
1. Copy the script to the target system
2. Run as Administrator for complete data collection
3. Execute from Command Prompt:

ir_windows.bat


#### Disclaimer: This tool is for authorized incident response and forensic investigations only. Use in compliance with all applicable laws and organizational policies.
